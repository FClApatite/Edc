This is an Electronic load project. It is capable of handling 50A 50V with 10/16 MOS tubes active,with an impedance less than 5mΩ and a stable rise time of 300ns.
To use the V2 Powerstage, Soldering 47R resistors on the G pin of mosfets is necessary,and do not forget to add an RC snubber to prevent oscillation caused by high dI/dt.
The V2.1 Powerstage is readily available ,and will appear in Rel 1.1
